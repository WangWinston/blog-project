## 3. 安全测试用例

### 3.1 认证安全

| 用例ID | 测试项 | 测试方法 | 预期结果 |
|--------|--------|----------|----------|
| SEC-001 | 无Token访问受保护API | 不带Token请求 /api/admin/** | 返回401 Unauthorized |
| SEC-002 | 过期Token访问 | 使用过期Token请求API | 返回401，提示Token过期 |
| SEC-003 | 篡改Token | 修改Token内容后请求 | 返回401，Token验证失败 |
| SEC-004 | 普通用户访问管理API | 使用USER角色Token访问 /api/admin/** | 返回403 Forbidden |
| SEC-005 | JWT有效期验证 | 检查Token有效期设置 | 有效期应为2小时 |

### 3.2 注入攻击防护

| 用例ID | 测试项 | 测试方法 | 预期结果 |
|--------|--------|----------|----------|
| SEC-010 | SQL注入-登录 | 用户名输入 `' OR '1'='1` | 登录失败，无数据泄露 |
| SEC-011 | SQL注入-搜索 | 搜索关键词输入SQL语句 | 正常搜索或提示错误，无注入 |
| SEC-012 | XSS攻击-评论 | 评论内容输入 `<script>alert(1)</script>` | 内容被转义或过滤，不执行脚本 |
| SEC-013 | XSS攻击-文章内容 | 文章内容包含恶意脚本 | 渲染时转义或过滤 |

### 3.3 其他安全测试

| 用例ID | 测试项 | 测试方法 | 预期结果 |
|--------|--------|----------|----------|
| SEC-020 | CORS配置 | 跨域请求测试 | 只允许配置的域名 |
| SEC-021 | 敏感信息泄露 | 检查API响应 | 不返回密码等敏感字段 |
| SEC-022 | 密码存储 | 检查数据库 | 密码使用BCrypt加密存储 |
| SEC-023 | 滑动验证码绕过 | 不完成验证直接提交 | 后端验证失败，拒绝请求 |

## 9. 附录：API端点清单

### 门户API (公开)
- `GET /api/portal/articles` - 文章列表
- `GET /api/portal/articles/{id}` - 文章详情
- `GET /api/portal/articles/hot` - 热门文章
- `GET /api/portal/categories` - 分类列表
- `GET /api/portal/tags` - 标签列表
- `GET /api/portal/comments/article/{id}` - 文章评论

### 认证API (公开)
- `POST /api/auth/login` - 登录
- `POST /api/auth/register` - 注册
- `GET /api/auth/github` - GitHub OAuth
- `GET /api/auth/github/callback` - OAuth回调

### 用户API (需登录)
- `POST /api/portal/user/articles/{id}/like` - 点赞
- `POST /api/portal/user/articles/{id}/favorite` - 收藏
- `GET /api/portal/user/favorites` - 收藏列表
- `POST /api/portal/comments` - 发表评论

### 管理API (需管理员权限)
- `GET/POST/PUT/DELETE /api/admin/articles` - 文章管理
- `POST /api/admin/articles/{id}/publish` - 发布文章
- `POST /api/admin/articles/{id}/schedule` - 定时发布
- `GET /api/admin/articles/{id}/versions` - 版本历史
- `GET/POST/PUT/DELETE /api/admin/categories` - 分类管理
- `GET/POST/PUT/DELETE /api/admin/tags` - 标签管理
- `GET /api/admin/comments` - 评论管理
- `GET /api/admin/dashboard/stats` - 统计数据
